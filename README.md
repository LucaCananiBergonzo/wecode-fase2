## :arrow_forward: Rodar o projeto

- Clone o repositório em sua máquina:
  ```sh
  $ git@github.com:LucaCananiBergonzo/wecode-fase2.git
  ```
- Instale o sass
  ```sh
  $ npm install -g sass
  ```
- Para observar o sass:
  ```sh
  $ sass --watch src/scss/style.scss:dist/assets/css/style.css
  ```
